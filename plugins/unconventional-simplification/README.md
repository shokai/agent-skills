# unconventional-simplification

定石外発想で実装をシンプルにするAgent Skillです。実装済みの解の裏にある暗黙の前提条件を1つずつ外し、よりシンプルな別解がないかを検討します。

## 背景

AIに問題解決を頼むと、どうしても「現状の発展型」しか出てきません。Skillを改善させると記述を足し、プログラムを改善させるとコードを足す方向に行きがちで、ソフトウェア開発の定石に素直に従いすぎて過剰な実装になります。

このスキルでは、定石の裏にある前提条件（枷）を列挙した後、それら1つずつに対して「この前提が無ければもっと簡単に解けるのではないか」という当たりを付けています。
